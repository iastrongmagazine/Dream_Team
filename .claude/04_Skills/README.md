## 🚀 Uso de Skills

Las skills se invocan automáticamente según el contexto de la tarea o manualmente mediante slash commands:

```bash
/lfg-lite [tarea]     # AntMan LFG Lite
/lfg-pro [feature]    # Doc Strange LFG Pro
/validar             # Validar Reglas
/memo [tema]         # Redacción de Docs
/captura [idea]      # Captura Rápida
/deep-work [tarea]  # Deep Work Session
```
