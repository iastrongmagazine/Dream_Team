# Invisible UI Design Guidelines

**Role:** Senior Product Designer
**Goal:** Strengthen hierarchy, focus, and navigation so the design disappears and data takes center stage.

## Core Principle

The UI must point to the data, not compete with it.

---

## 1. Primary Focus

- **Identify the One Thing:** Find the single most important knowledge or decision supported by this screen.
- **Visual Dominance:** Make that element visually dominant at first glance.
- **Support or Yield:** All other elements must support it or yield to it.

## 2. Navigation & Sidebar Audit (Critical)

**Purpose:** Global/Local navigation, utilties, context. If unclear → Remove or Consolidate.

### Visual Weight

- Lower contrast.
- Light typography.
- Minimal iconography.
- Sidebars must **frame**, not compete.

### Priority

- **Eliminate** redundant elements.
- **Group** related actions.
- **Highlight** current location subtly.

### Discoverability vs Noise

- Essentials always visible.
- Secondary → Progressive disclosure.

## 3. Navigation Discipline

- Separate Global from Local navigation.
- Navigation guides, it does not decorate.

## 4. Color & Focus strategy

- **Base:** Neutral palette.
- **Accents:** Use sparingly.
- **System Colors:** Only for real states (Success, Error, Alert).

## 5. Visual Restraint

- Identify and remove unnecessary decorative elements.
- De-emphasize secondary information.
- Avoid multiple focal points.

## 6. The 3-Second Test

In a 3-second glance, the user must know:

1. **What matters.**
2. **Where to look.**
3. **What action is next.**

## 7. Metric Hierarchy & "North Star"

Prioritize metrics hierarchically. Use highlighting or positional dominance (e.g., top-left, larger size) for the **Main Functionality**.

- **Main Functionality:** Represents key user behavior. It indicates habit, retention, and value.
- **Secondary Info:** Move to the right or de-emphasize.

---

## Summary Checklist

- [ ] **Data First:** Is the data the boldest thing on the page?
- [ ] **Invisible Nav:** Does the sidebar fade into the background?
- [ ] **North Star:** Is the key metric instantly visible (3-second test)?
- [ ] **Neutral Base:** Are colors used for function/signal, not decoration?
