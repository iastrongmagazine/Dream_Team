# Analytics Workflow 📊

Skill para el análisis profundo de datos, visualización de flujos y generación de reportes de cohortes automatizados.

## 🎯 Objetivo

Transformar datos crutos en insights accionables mediante el uso de algoritmos de agrupamiento, análisis temporal y visualización dinámica.

## 🛠️ Herramientas y Scripts

- **`detailed_cohort_analysis.py`**: Análisis de retención y comportamiento de grupos de usuarios.
- **`data_visualization_flows.py`**: Generación de diagramas de flujo y mapas de calor de interacción.
- **`data_analysis_skills.py`**: Utilidades generales de limpieza y procesamiento estadístico.

## 🔄 Workflow

1.  **Ingesta**: Carga de fuentes CSV/JSON.
2.  **Procesamiento**: Limpieza y normalización de esquemas.
3.  **Análisis**: Ejecución de modelos de cohortes o tendencias.
4.  **Reporte**: Generación de dashboards estáticos en Markdown.

---

> [!NOTE]
> Esta skill está optimizada para integrarse con la Skill de **CSV Management** para el manejo de grandes volúmenes de datos.
