# 📈 Silicon Valley Performance

Esta Skill eleva la simple analítica de datos a una función de consultoría estratégica. No se limita a mostrar números; interpreta tendencias, identifica riesgos y propone acciones de escalado utilizando los estándares de eficiencia de las Big Tech.

## 🚀 Triggers

- "Analiza este CSV de ventas y dime qué acciones tomar."
- "Genera un reporte de rendimiento del sistema PersonalOS."
- "¿Cómo está el consumo de tokens y el presupuesto este mes?"
- "Realiza un análisis de cohortes sobre estos datos de usuario."

## 🛠️ Workflow: The Performance Way

### 1. Ingesta Inteligente (Smart Ingestion)
El sistema detecta automáticamente el dominio del set de datos:
- **Metodología**: Utiliza `08_Scripts_Os/20_Master_Analytics_Factory.py` para perfilar datos y detectar dominios (SaaS, E-commerce, Marketing, Ops).
- **Workspace**: Crea un entorno de análisis aislado para evitar contaminación.

### 2. Generación de Insights (Deep Context Analysis)
Integración con el sistema de reporte consolidado:
- **AIPM Insight (`30_AIPM_Consolidated_Report.py`)**: Extrae métricas de salud, riesgos y recomendaciones de todas las herramientas AIPM.
- **Narrativa Ejecutiva**: Transforma traces técnicos en "Ladrillos de Contexto" accionables.

### 3. Visualización Premium (Executive Dashboard)
- Generación de reportes en Markdown con tablas comparativas y matrices de decisión.
- Branding de **PersonalOS** en cada reporte generado.
- Notificaciones de voz para resaltar los hallazgos críticos.

## 📋 Instrucciones de Ejecución

1. **Escanear**: Determinar la fuente de datos (Traces AIPM o Archivos Externos).
2. **Procesar**: Ejecutar la factoría de analítica para generar scripts de análisis dinámicos.
3. **Interpretar**: Añadir la capa de sabiduría "Silicon Valley" (¿Por qué importa esto?).
4. **Presentar**: Entregar el reporte final y resumir verbalmente los 3 puntos clave.

> [!TIP]
> Combina esta Skill con `Elite Agent Auditor` para obtener una visión 360° de eficiencia y calidad.
