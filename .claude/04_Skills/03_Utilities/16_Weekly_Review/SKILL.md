# Weekly Review Skill

## When to use this skill

- Friday afternoons or Sunday evenings.
- When the user asks "How did my week go?"
- When planning the next week's workload.

## Workflow

1.  **Harvest Progress**: Analyze completed tasks in `03_Task/` and `README.md`.
2.  **Evaluate Goals**: Check progress against `GOALS.md`.
3.  **Identify Blockers**: List all stalled or blocked tasks.
4.  **Plan Future**: Propose P0/P1 tasks for the next week.
5.  **Budget Time**: Estimate deep work capacity vs. meetings.

## Instructions

- Be objective and data-driven.
- Highlight wins but also call out stalled tasks honestly.
- Encourage updating `GOALS.md` if priorities have shifted.
