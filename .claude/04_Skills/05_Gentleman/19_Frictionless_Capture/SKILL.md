# Frictionless Capture Skill

## When to use this skill

- When the user has a fleeting idea or learning.
- When finding a useful link or snippet.
- Triggered by "Process my capture" or "New insight."

## Workflow

1.  **Raw Capture**: Append input to `BACKLOG.md` immediately.
2.  **Classification**: Distinguish between an **Insight** (Knowledge) or an **Action** (Task).
3.  **Refinement**: Suggest appropriate home in `01_Brain/` or create a task in `02_Operations/01_Active_Tasks/`.

## Instructions

- Prioritize speed. Don't worry about formatting during the initial capture.
- Use `zinking-transform` for later refinement.
