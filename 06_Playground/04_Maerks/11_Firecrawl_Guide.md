## 🔥 ¿Qué es Firecrawl?

**Definición simple:**
> "Put in a website → goes through Firecrawl API → get back clean markdown, structured JSON, screenshots → feed to any AI model"

**Es el AWS para datos web:**
- Antes: comprar servidores, gestionar proxies, manejar anti-bot
- Ahora: 1 llamada API y listo

## 🎯 Framework: Pick Niche → Build → Package → Sell

### Step 1: Pick Niche
- ¿Qué datos pagan las personas en esta industria?
- Nichos ejemplos:
  - Sneakers (StockX, GOAT, eBay)
  - Dentistas (competidores SEO)
  - Remote AI/ML jobs (career pages)
  - Crypto tokens (due diligence)
  - Amazon FBA (product gaps)

### Step 2: Build
- Usar Firecrawl Agent
- Python script simple
- O Cloud Code / Cursor

### Step 3: Package
- CSV output
- Dashboard
- Slack alerts
- API

### Step 4: Sell
- No vender la herramienta → vender el OUTPUT
- $500 - $5,000/month por cliente
- Automatizar y escalar
