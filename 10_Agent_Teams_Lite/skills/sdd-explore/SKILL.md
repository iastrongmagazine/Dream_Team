# SDD Explore

Explora ideas, investiga el codebase y clarifica requisitos antes de crear una propuesta.

## Proceso

1. **Entender la intención del usuario**: Qué quiere lograr
2. **Investigar el codebase**: Análisis de código existente
3. **Identificar constraints**: Limitaciones técnicas, de tiempo, de scope
4. **Generar opciones**: Múltiples enfoques posibles

## Outputs

- `explore/{change-name}/explore.md` - Hallazgos de investigación
- Lista de opciones con tradeoffs
- Recomendación inicial
