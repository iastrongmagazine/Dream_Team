# 🧹 Higiene y Seguridad - Cocina - Macano

## Normas de Higiene

### Personal
- [ ] Manos lavadas cada 30 min
- [ ] Uñas cortas y limpias
- [ ] Cabello recogido
- [ ] Guantes cambiadas entre tareas
- [ ] Delantal limpio

### Área de Trabajo
- [ ] Superficies desinfectadas
- [ ] Utensilios limpios
- [ ] Productos separados (crudo/cocido)
- [ ] Temperatura de refrigerador <4°C
- [ ] Congelador < -18°C
