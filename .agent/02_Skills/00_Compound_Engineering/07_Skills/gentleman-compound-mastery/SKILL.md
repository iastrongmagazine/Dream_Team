# 🔥 Gentleman + Compound Engineering Mastery

Este documento establece el sistema integrado de desarrollo más potente del mundo, combinando Gentleman (manos que codifican) con Compound Engineering (cerebro que revisa).

---

## 🎯 Cuándo Usar Este Sistema

- **Desarrollo diario** de features
- **Code review** profesional
- **Investigación** de mejores prácticas
- **Planificación** de proyectos complejos
- **Optimización** de workflows

## 🎯 Cómo Usar el Sistema

### 1. Para Nuevos Proyectos

```
1. /ce:ideate → identificar mejoras
2. /ce:brainstorm → explorar enfoque
3. /ce:plan → crear plan
4. [Gentleman Framework Skill] → codificar
5. /ce:review → revisar con CE agents
6. /ce:compound → documentar
```

### 2. Para Code Review

```
1. Usar: /ce:review
2. O usar agents específicos:
   - security-sentinel (seguridad)
   - performance-oracle (performance)
   - kieran-typescript-reviewer (TS)
   - architecture-strategist (arquitectura)
```

### 3. Para Investigación

```
1. Usar: best-practices-researcher
2. O: framework-docs-researcher
3. O: learnings-researcher
```

### 4. Para Diseño

```
1. Usar: Taste skills (04_Product_Design)
2. Para Figma: figma-design-sync (CE)
3. Para iterar: design-iterator (CE)
```
