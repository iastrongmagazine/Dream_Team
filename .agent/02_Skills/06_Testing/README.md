# Testing

## Purpose
Comprehensive testing skills covering TDD, debugging, verification, and quality assurance across multiple frameworks and methodologies.

## Skills Included
| # | Skill | Trigger | Description |
|---|-------|---------|-------------|
| 01 | Test-Driven Development | "tdd", "test first" | Red-Green-Refactor cycle for quality |
| 02 | Systematic Debugging | "debug", "debugging" | 4-phase methodology for root cause analysis |
| 03 | Verification Before Completion | "verify", "qc", "quality check" | Exhaustive quality control before completion |
| 04 | Verify and Commit | "verify and commit" | Verification + atomic commit workflow |
| 05 | Test Resource Management | "test resources", "fixtures" | Test fixtures and resource management |
| 06 | Testing Coverage | "coverage", "test coverage" | Coverage analysis and gap identification |
| 07 | Go Testing | "go test", "golang testing" | Go testing with Bubbletea patterns |
| 08 | TUI Quality | "tui testing", "bubbletea" | TUI quality rules for Bubbletea/Lipgloss |
| 09 | E2E Testing | "e2e", "end to end" | End-to-end testing with Playwright |
| 10 | Integration Testing | "integration test", "api testing" | Multi-component system testing |
| 11 | Test Coverage | "coverage analysis" | Coverage optimization strategies |
| 12 | Edge Case | "edge case", "boundary testing" | Boundary analysis and failure modes |
| 13 | Evaluation | "evaluation", "benchmark" | Agent and system evaluation frameworks |
| 14 | Skill Testing Automation | "test skill", "automation" | Automated skill testing |
