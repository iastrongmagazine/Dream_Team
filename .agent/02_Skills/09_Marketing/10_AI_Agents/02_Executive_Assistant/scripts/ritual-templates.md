# Ritual Templates

## Daily Rituals

### Morning Routine (10 min)

```
1. Review calendar for today
2. Check for urgent messages
3. Identify P0 items
4. Block deep work time
5. Set intentions for the day
```

### End-of-Day Review (5 min)

```
1. What did I accomplish?
2. What's carryover to tomorrow?
3. Any follow-ups needed?
4. What's my plan for tomorrow?
```

## Weekly Rituals

### Monday Planning (15 min)

```
1. Review week ahead
2. Identify critical meetings/deadlines
3. Set 3 priorities for the week
4. Clear any carryover from last week
```

### Friday Reflection (10 min)

```
1. What went well?
2. What didn't go as planned?
3. What did I learn?
4. What do I want to do differently?
```

### Weekly Sync (15 min)

```
1. Accomplishments (last week)
2. Priorities (this week)
3. Blockers
4. Help needed
```

## Monthly Rituals

### Monthly Review (30 min)

```
1. Review month goals
2. Assess progress
3. Identify themes
4. Set priorities for next month
5. Celebrate wins
```

### Monthly Planning (20 min)

```
1. Review quarterly objectives
2. Identify key initiatives for month
3. Block time for important work
4. Schedule critical meetings
```

## Meeting Rituals

### One-on-One (30 min)

```
Pre-meeting (5 min):
- Review notes from last meeting
- Identify topics to discuss

Meeting structure:
- Check-in: How are things? (2 min)
- Updates: What's new? (5 min)
- Discussion: Topics (15 min)
- Action items: What next? (5 min)
- Feedback: How can I help? (3 min)

Post-meeting:
- Send summary
- Track action items
```

### Team Meeting (60 min)

```
Pre-meeting:
- Collect async updates
- Prepare agenda
- Time-box each topic

Meeting structure:
- Quick wins (5 min)
- Updates (10 min)
- Deep dive (30 min)
- Decisions (10 min)
- Next steps (5 min)

Post-meeting:
- Send recording/summary
- Track action items
- Update project tracker
```
