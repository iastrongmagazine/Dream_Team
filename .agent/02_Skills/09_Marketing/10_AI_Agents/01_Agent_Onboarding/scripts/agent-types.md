# Common Agent Types

## Categorization by Capability Level

### Level 1: Task-Specific Agents
Single-purpose, narrow scope. Great for repetitive, well-defined tasks.

**Examples**:
- PDF processor
- Image resizer
- Data entry assistant

**Characteristics**:
- Clear input/output definition
- Limited flexibility needed
- High volume, low complexity

### Level 2: Domain Specialists
Expert in a specific domain with context awareness.

**Examples**:
- Marketing copywriter
- Technical documentation writer
- Financial analyst

**Characteristics**:
- Deep domain knowledge
- Context-dependent decisions
- Moderate complexity

### Level 3: Strategic Advisors
High-level thinking, business context, recommendations.

**Examples**:
- Business strategy consultant
- Product manager
- CMO role

**Characteristics**:
- Cross-functional knowledge
- Ambiguous problems
- Low frequency, high impact

### Level 4: Orchestrators
Coordinate other agents or tools, manage workflows.

**Examples**:
- Team lead agent
- Project manager
- DevOps coordinator

**Characteristics**:
- Multi-agent coordination
- Complex workflow management
- Error handling and recovery

---

## Choosing the Right Level

| Use Case | Recommended Level |
|----------|-------------------|
| Repetitive, well-defined tasks | Level 1 |
| Domain expertise needed | Level 2 |
| Strategic decisions | Level 3 |
| Multi-agent coordination | Level 4 |

**Start simple**: Always begin with Level 1. Move up only when complexity demands it.
