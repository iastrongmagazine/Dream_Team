# Excalidraw Tips

## Keyboard Shortcuts
- `V` - Select
- `R` - Rectangle
- `E` - Ellipse
- `A` - Arrow
- `T` - Text
- `D` - Diamond
- `H` - Hand (pan)

## Styling
- Stroke width: 1-2 for most elements
- Fill: Use opacity 10-20%
- Font: Hand-written for sketch feel

## Organization
- Group related elements (Ctrl+G)
- Align using guidelines
- Distribute evenly

## Export
- PNG: For sharing, presentations
- SVG: For web, scaling
- .excalidraw: Source file for editing

## Color Palette (PersonalOS)
- Primary: #3B82F6 (blue)
- Secondary: #10B981 (green)
- Accent: #F59E0B (amber)
- Neutral: #6B7280 (gray)
- Background: #FFFFFF
