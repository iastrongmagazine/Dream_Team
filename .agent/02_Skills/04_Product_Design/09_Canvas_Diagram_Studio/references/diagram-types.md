# Diagram Types

## Architecture Diagrams
- Components as boxes
- Arrows for data flow
- Labels for each component
- Legend for colors

## Mind Maps
- Central idea in middle
- Branches radiating out
- Sub-branches for details
- Color-code categories

## Wireframes
- Low fidelity initially
- Key screens only
- Annotations for interactions
- Mobile + Desktop variants

## Conceptual Schemes
- Entities as shapes
- Relationships as lines
- Direction indicated by arrows
- Group related items

## Flowcharts
- Start/End as ovals
- Process as rectangles
- Decision as diamonds
- Flow direction: top to bottom or left to right
