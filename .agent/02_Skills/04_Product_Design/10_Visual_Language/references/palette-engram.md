# Engram Palette

## TUI-Inspired Colors

### Background
- `--bg-primary`: #0a0a0a (deep black)
- `--bg-secondary`: #18181b (zinc-900)
- `--bg-tertiary`: #27272a (zinc-800)

### Text
- `--text-primary`: #fafafa (zinc-50)
- `--text-secondary`: #a1a1aa (zinc-400)
- `--text-muted`: #71717a (zinc-500)

### Accents (TUI-inspired)
- `--accent-primary`: #22d3ee (cyan-400)
- `--accent-secondary": #a78bfa (violet-400)
- `--accent-success`: #34d399 (emerald-400)
- `--accent-warning`: #fbbf24 (amber-400)
- `--accent-danger`: #f87171 (red-400)

### Mono/Display
- `--mono-color`: #22d3ee (cyan for code/mono)
- `--display-color`: #a78bfa (violet for display text)

## Usage
- Use mono accent for code blocks, timestamps
- Use display accent for hero text, highlights
- Keep backgrounds dark for TUI feel
