# Visual Rules

## Core Principles

1. **Engram Identity**
   - Must feel like Engram, not generic SaaS
   - TUI-inspired aesthetic
   - Distinct from admin panels

2. **Hierarchy**
   - Strong visual hierarchy
   - Clear primary/secondary/tertiary levels
   - Headings stand out

3. **Containers**
   - Fewer containers, not more
   - Each container has purpose
   - Group related content only

4. **Breathing Room**
   - Generous whitespace
   - Clear margins and padding
   - Content not cramped

5. **Accents**
   - TUI-inspired mono accents
   - Display typography intentional
   - Use sparingly for emphasis

## Typography Scale
- Display: 32-48px, tight tracking
- Heading 1: 24-28px
- Heading 2: 20-24px
- Body: 16px
- Small: 14px
- Caption: 12px

## Spacing Scale
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
