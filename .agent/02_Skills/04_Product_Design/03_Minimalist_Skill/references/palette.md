# Minimalist Palette

## Warm Monochrome Base

### Backgrounds
| Element | Color | Hex |
|---------|-------|-----|
| Page BG | Warm White | #FAF9F7 |
| Card BG | Pure White | #FFFFFF |
| Hover | Light Gray | #F5F5F4 |

### Text
| Element | Color | Hex |
|---------|-------|-----|
| Heading | Charcoal | #1C1917 |
| Body | Off-black | #2F3437 |
| Secondary | Muted | #787774 |
| Disabled | Light Gray | #A8A29E |

### Muted Pastels (Accents)
| Name | Hex | Use For |
|------|-----|---------|
| Sage | #E8E5E0 | Backgrounds, tags |
| Blush | #F5E6E0 | Subtle highlights |
| Sand | #EDE8D5 | Warm accents |
| Stone | #D6D3D1 | Borders, dividers |

## Rules
1. NO gradients
2. NO heavy shadows
3. NO bright colors
4. NO neon anything
5. Max 2-3 pastels total
6. Keep everything warm
