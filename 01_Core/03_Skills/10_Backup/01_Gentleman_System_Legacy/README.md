# 05_Gentleman - Centro de Organización de Skills

## Propósito
Centralizar todas las skills del ecosistema Gentleman bajo una estructura numerada y jerárquica.

```

├── 02_Work/          # Implementación técnica
├── 03_Review/        # Calidad y revisión
└── 04_Compound/      # Ecosistema Compound
```
