🚀 INSTALACIÓN RÁPIDA (3 PASOS)
════════════════════════════════════════════════════════════════════════════

1️⃣ Instalar dependencias del sistema:

   Ubuntu/Debian:
   sudo apt-get install tesseract-ocr tesseract-ocr-spa poppler-utils

   macOS:
   brew install tesseract tesseract-lang poppler

2️⃣ Instalar dependencias Python:

   pip install -r requirements.txt --break-system-packages

   O automático:
   bash install.sh

3️⃣ ¡Listo! Ejecutar:

   python invoice_processor.py ./tus_facturas


⚡ USO BÁSICO
════════════════════════════════════════════════════════════════════════════

python dashboard.py

🆘 SOLUCIÓN DE PROBLEMAS
════════════════════════════════════════════════════════════════════════════

❌ "Tesseract not found"
   → Instalar tesseract-ocr (ver instalación arriba)

❌ "pdf2image requires poppler"
   → Instalar poppler-utils

❌ Muy lento con OCR
   → Reducir workers: -w 2
   → Desactivar OCR: --no-ocr
   → Reducir páginas OCR (editar código línea ~185)

❌ Memoria insuficiente
   → Reducir workers
   → Procesar en lotes
