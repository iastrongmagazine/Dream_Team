# Design Workflow

## Step 1: Conceptualization
- Define architecture visual
- Identify key screens
- Map user flows
- Reference Manifesto de Diseño

## Step 2: AI Generation
- Use `generate_screen` for base structures
- Create multiple variations
- Iterate on initial concepts

## Step 3: Refinement
- Open in Infinite Canvas
- Adjust layouts
- Refine spacing and typography
- Add micro-interactions

## Step 4: Style Validation
- Check HSL values
- Verify Glassmorphism (if used)
- Confirm spacing consistency
- Test color contrast

## Step 5: Implementation
- Use `export_to_code`
- Review generated code
- Integrate into codebase
- Add state and logic

## Quality Checklist
- [ ] Design matches brand guidelines
- [ ] Responsive considered
- [ ] Accessibility checked
- [ ] Code is clean and reusable
