# Typography System

## Primary Sans-Serif (Body, UI, Buttons)
- SF Pro Display
- Geist Sans
- Helvetica Neue
- Switzer

## Editorial Serif (Hero Headings & Quotes)
- Lyon Text
- Newsreader
- Playfair Display
- Instrument Serif

Apply:
- letter-spacing: -0.02em to -0.04em
- line-height: 1.1

## Monospace (Code, Meta-data)
- Geist Mono
- SF Mono
- JetBrains Mono

## Text Colors
- Body: NOT #000000 — use #111111 or #2F3437
- Secondary: #787774 (muted gray)
- line-height: 1.6 for body

## Rules
- Extreme typographic contrast
- Editorial feel
- Premium fonts only
