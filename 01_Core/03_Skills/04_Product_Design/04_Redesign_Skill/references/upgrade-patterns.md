# Upgrade Patterns

## Typography Upgrade
```css
/* Before */
font-family: 'Inter', sans-serif;
font-size: 24px;

/* After */
font-family: 'Geist Sans', sans-serif;
font-size: 32px;
letter-spacing: -0.03em;
line-height: 1.1;
```

## Shadow Upgrade
```css
/* Before */
box-shadow: 0 4px 6px rgba(0,0,0,0.1);

/* After */
box-shadow: 0 1px 2px rgba(0,0,0,0.04);
```

## Color Upgrade
```css
/* Before */
background: #000000;
color: #ffffff;

/* After */
background: #faf9f7;
color: #1c1917;
```

## Spacing Upgrade
```css
/* Before */
gap: 8px;
padding: 16px;

/* After */
gap: 24px;
padding: 32px;
```

## Border Upgrade
```css
/* Before */
border: 1px solid #e5e7eb;

/* After */
border: 1px solid rgba(0,0,0,0.08);
```
