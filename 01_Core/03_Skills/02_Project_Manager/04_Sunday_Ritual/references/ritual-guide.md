# Sunday Ritual Guide

## Overview

The Sunday Ritual is a 30-45 minute weekly reset to:
1. Review what happened this week
2. Clear accumulated backlog
3. Plan next week
4. Maintain the system

## Timing

Best time: Sunday evening (before Monday)
Alternative: Friday end of day

## Steps

### 1. Reflective Harvest (10 min)

Ask yourself:
- What did I accomplish this week?
- What got blocked?
- What did I learn?

### 2. Capture Sweep (10 min)

Run backlog processing:
- Review all items in BACKLOG.md
- Convert to tasks or archive
- Check for duplicates

### 3. Architecture (15 min)

Plan next week:
- Select max 3 P0 tasks
- Align with GOALS.md
- Estimate time required

### 4. System Maintenance (10 min)

- Archive completed tasks
- Sync notes to knowledge base
- Run health checks

## Tips

- Do it same time each week to build habit
- Block 45 minutes minimum
- Do it before planning next week
- Review last week's ritual notes
