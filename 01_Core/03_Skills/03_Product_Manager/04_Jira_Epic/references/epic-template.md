# Epic Template

## Format

```markdown
# [EPIC] {Title}

**Figma:** {link}

## Feature Overview
{2-3 paragraphs: what, who, why}

## Requirements

### {Area 1}
- Requirement 1
- Requirement 2

### {Area 2}
- Requirement 1

## Technical Considerations

### Performance
- {requirement}

### Data Integration
- {source}

### UI Components
- {component}

## Implementation Checklist
- [ ] Deliverable 1
- [ ] Deliverable 2

## Diagrams
```mermaid
graph TB
    A --> B
```
```

## Title Convention
`[EPIC] Feature Name`

Examples:
- `[EPIC] Findings View`
- `[EPIC] Multi-tenant Support`
