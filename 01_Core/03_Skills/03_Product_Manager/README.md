# Product Manager

## Purpose
Strategic planning and product management workflow. Covers ideation, planning, execution, and documentation for product features and improvements.

## Skills Included
| # | Skill | Trigger | Description |
|---|-------|---------|-------------|
| 01 | Brainstorming | "brainstorm", "idear", "lluvia de ideas" | Explore intentions, requirements, and alternative approaches |
| 02 | Writing Plans | "write plan", "crear plan" | Structured task planning using ShipKit framework |
| 03 | Executing Plans | "execute plan", "ejecutar plan" | Execute pre-written action plans with traceability |
| 04 | Jira Epic | "create epic", "jira epic" | Create Jira epics for large features |
| 05 | Jira Task | "create task", "jira task" | Create Jira tasks following Prowler format |
| 06 | Branch PR | "branch PR", "pull request" | PR creation workflow with issue-first enforcement |
| 07 | Strategy Memos | "strategy memo", "memo estratégico" | FocusFlow framework for strategic decisions |
| 08 | AI Task Planning | "plan with AI", "planificar tareas" | AI-assisted structured planning |

## Usage
```
"let's brainstorm {idea}"   # Explore and clarify requirements
"write a plan for {feature}" # Create structured implementation plan
"execute the plan"           # Run pre-defined plan
"create a Jira epic"         # Generate epic for large feature
"create a Jira task"         # Generate task ticket
"open a PR"                  # Create pull request workflow
"write a strategy memo"     # Document strategic decision
```
