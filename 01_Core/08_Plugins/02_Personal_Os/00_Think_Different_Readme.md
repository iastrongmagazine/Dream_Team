# Think Different PersonalOS Plugin

Plugin para Claude Code que trae el sistema completo de productividad personal Think Different.

## Características

- **Skill PersonalOS** - Activa todo el sistema de productividad
- **SDD Workflow** - Metodología de desarrollo estructurado
- **System Guardian** - Validación automática del proyecto

## Instalación

```bash
/plugin install https://github.com/iastrongmagazine/Dream_Team
```

## Uso

### Activar PersonalOS
```
/think-different:personal-os
```

### SDD Workflow
```
/think-different:sdd-workflow
```

### System Guardian
```
/think-different:system-guardian
```

## Requisitos

- Claude Code CLI
- Git instalado
- Python 3.8+ (para algunos scripts)
