# exa

> **Categoría:** Research (Tier 4)
> **Uso:** Búsqueda web con IA

## Descripción

Motor de búsqueda web potenciado por IA.

## Uso Principal

```bash
# Búsqueda simple
query: "latest React 19 features"

# Búsqueda con filtros
category: "technical"
num_results: 10
```

## Configuración

```json
"exa": {
  "command": "npx.cmd",
  "args": ["-y", "exa-mcp-server"],
  "env": {
    "EXA_API_KEY": "..."
  }
}
```

## Casos de Uso

- 🔍 Búsqueda web
- 📚 Research de tecnologías
- 📰 Noticias del sector
