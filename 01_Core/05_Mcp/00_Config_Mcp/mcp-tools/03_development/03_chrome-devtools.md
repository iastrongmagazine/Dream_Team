# chrome-devtools

> **Categoría:** Development (Tier 3)
> **Uso:** Chrome browser automation

## Descripción

Integración con Chrome DevTools para automation de browser.

## Uso Principal

```bash
# Abrir inspector
chrome-devtools: inspect

# Ejecutar script
chrome-devtools: evaluate "console.log('test')"
```

## Configuración

```json
"chrome-devtools": {
  "command": "npx.cmd",
  "args": ["-y", "chrome-devtools-mcp@latest"]
}
```

## Casos de Uso

- 🌐 Browser automation
- 🐛 Debugging web
- 📸 Screenshots
