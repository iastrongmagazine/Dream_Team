# aim-memory-bank

> **Categoría:** Knowledge (Tier 2)
> **Uso:** Knowledge graph y memoria estructurada

## Descripción

Sistema de memory graph para organizar conocimiento en grafos.

## Uso Principal

```bash
# Agregar al knowledge graph
concept: "Agentic Engineering"
connections: ["LLM", "Tools", "Prompts"]

# Buscar relaciones
query: "engineering patterns"
```

## Configuración

```json
"aim-memory-bank": {
  "command": "npx.cmd",
  "args": ["-y", "mcp-knowledge-graph", "--memory-path", ".aim"]
}
```

## Casos de Uso

- 🕸️ Knowledge graphs
- 🔗 Conexiones entre conceptos
- 📊 Memoria estructurada
