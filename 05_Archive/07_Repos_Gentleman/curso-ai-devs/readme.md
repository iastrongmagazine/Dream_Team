# AI Tools Para desarrolladores

En este repo encontrarás una lista de herramientas de IA que pueden ayudarte en tu desarrollo.

Es el Repo que usaremos para el curso de AI Tools Para desarrolladores. Tiene un proyecto llamado Platzi FC y un archivo ai-tools.md con las herramientas que vamos a usar.

La idea es que este archivo pueda recibir colaboraciones de la comunidad para agregar más herramientas.
