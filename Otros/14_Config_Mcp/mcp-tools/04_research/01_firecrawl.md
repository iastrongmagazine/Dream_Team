# firecrawl-mcp

> **Categoría:** Research (Tier 4)
> **Uso:** Web scraping y data extraction

## Descripción

Motor de scraping avanzado para extraer datos estructurados de páginas web.

## Uso Principal

```bash
# Scraping básico
url: "https://stripe.com/blog"

# Extracción estructurada
format: "markdown"
```

## Configuración

```json
"firecrawl-mcp": {
  "command": "npx.cmd",
  "args": ["-y", "firecrawl-mcp"],
  "env": {
    "FIRECRAWL_API_KEY": "fc-..."
  }
}
```

## Casos de Uso

- 🕷️ Web scraping
- 📊 Data extraction
- 🔬 Research de competidores
