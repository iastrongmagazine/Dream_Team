# Playwright

> **Categoría:** Development (Tier 3)
> **Uso:** Testing E2E automatizado

## Descripción

Framework de testing E2E para aplicaciones web.

## Uso Principal

```bash
# Ejecutar tests
npx playwright test

# Abrir UI
npx playwright codegen
```

## Configuración

```json
"Playwright": {
  "command": "npx.cmd",
  "args": ["@playwright/mcp@latest"]
}
```

## Casos de Uso

- 🧪 Tests end-to-end
- 🔍 Browser automation
- 📱 Testing responsive
