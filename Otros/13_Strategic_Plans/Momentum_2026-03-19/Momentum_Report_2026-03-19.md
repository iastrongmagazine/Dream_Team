## 🎓 LESSONS LEARNED

1. **Hardcoded paths = technical debt** → Always use `config_paths.py`
2. **Embedded git repos = pain** → Use bare mirrors for backup
3. **UTF-8 encoding on Windows** → Configure `PYTHONIOENCODING=utf-8`
4. **Tests are not optional** → Every fix needs a test
5. **RTM + Edge Cases = preventive medicine** → Document before breaking
