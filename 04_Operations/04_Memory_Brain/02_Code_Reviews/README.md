## Cómo Generar un Code Review

Después de una sesión o plan significativo:

```
"Hacé el double code review de esta sesión"
```

El sistema ejecutará:
1. FASE 1: Planning Status
2. FASE 2: 6 Sombreros (🔵🔴🟡🟢🟣⚪)
3. FASE 3: Verification Check

Y guardará en:
```
00_Code_Reviews/YYYY-MM-DD_[nombre]_review.md
```
