## Double Diamond

### What It Solves
Framework for comprehensive problem-solving: discover → define → develop → deliver.

### When to Use
- Complex problems requiring research
- New product development
- Service design
- Strategic initiatives

### How It Works

```
    DISCOVER          DEFINE           DEVELOP         DELIVER
    (divergent)      (convergent)     (divergent)     (convergent)
    
    ┌─────────┐      ┌─────────┐      ┌─────────┐      ┌─────────┐
    │         │      │         │      │         │      │         │
    │  Research│ ──► │  Synth  │ ──► │  Build  │ ──► │  Launch│
    │         │      │         │      │         │      │         │
    └─────────┘      └─────────┘      └─────────┘      └─────────┘
    
    Explore         Focus            Explore          Focus
    Many ideas      One direction   Many options     One solution
```

### Phases

| Phase | Activities | Output |
|-------|-------------|--------|
| **Discover** | User research, interviews, observation, benchmarking | Insights, pain points |
| **Define** | Synthesis, personas, problem statements, journey maps | Clear brief |
| **Develop** | Ideation, prototyping, testing iterations | Solutions |
| **Deliver** | Implementation, launch, monitoring | Live product |

### Example
```
Discover: 20 user interviews → "Users hate password resets"
Define: Problem statement → "Users need password reset in <30 seconds"
Develop: 3 prototypes, test each → "Biometric + SMS fallback"
Deliver: Launch in 2 weeks → NPS +15
```

### When NOT Use
- Quick fixes
- Already well-understood problems
- Pure technical projects

### Integration with AI (2026)
- AI-enhanced Double Diamond uses LLM for synthesis
- Agentic UX: AI suggests based on patterns
